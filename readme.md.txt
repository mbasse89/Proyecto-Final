Mi nombre es Matias Basse, creador de la página web, Yerba Mate Experience.

Este proyecto se realizo en el contexto del curso de desarrollo Web en CoderHouse. Los objetivos del mismo
eran crear una página web con un minimo de 5 htmls. En estos se tendrían quie navegar entre sí sin problemas
manteniendo la coherencia de la web utilizando las distintas herramientas aprendidas durante la cursada. El proyecto 
debe de ser subido a GitHub mediante la herramient GIT, donde pudimos ver la creación de repositorios. Indagamos en el uso
de Box Model para poder mover los contenedores, Flex para poder manipular la ubicación de estos últimos.
Durante el curso aprendimos el uso de grillas ya sea con Grid o utilizando la libreria Boostrap. La instalación de Node para utilzar
la compilación del código creado a través de Sass para poder estilizar el diseño web. 

	Primero se comenzó con el diseño de Wireframes para empezar a maquetar los primeros bocetos de la web. Con la utlización
de Box Model empezamos a jugar con la estilización de la web y sus contenedores. Al incorporar Flex se le añadió dinamismo, 
ya que de a poco empezó a dejar de ser estático todo el diseño. Grid fue otra herramienta que complementó Flex, ya que es otra
manera de ordenar el contenido. Luego de aprender estos 2 métodos nos enseñaron Boostrap, un framework muy utilzado, en el que personalmente
incorporé fácilmente por su simplicidad. 
	
	Otra de las herramientas que fue escencial en este proyecto fue GIT que con la creación de repositorios locales, pudimos ir 
guardando nuestro proyecto. A esto le adherimos GitHub que se sumó a GitHub, creando en este caso repositorios remotos y pudiendo 
crear un Deploy donde poder compartir nuestro proyecto. 

	Una vex que tuvimos el trabajo avanzado utilzamos Sass, donde debimos de trasladar nuestro archivo Css a este nuevo lenguaje, 
que nos ayudó a reorganizar nuestro código. La creación de variables, partiales, mixins y extends nos dió otra perspectiva de como
trabajar en futuros proyectos.

	Uno de los objetivos más importantes a lograr en este proyecto, fue el de poder de lograr una página web responsiva, es decir un 
sitio que pueda verse en diferentes dispositivos móbiles. Para lograr esto, através de breakpoints se fue modificando 
el tamaño de imagenes, fuentes y movilizar los contenedores para que queden adaptables. 

	En el transcurso de la cursada los tutores nos fueron relevando el proyecto, dandonos indicaciones y corrigiendo los errores cometidos. A la vez se optó por recurrir a material externo, ya sean videos y tutoriales para la mejor comprensión de los temas brindados en clase. 

	Uno de los últimos temas vistos fue la de servidores y SEO. Hemos visto reglas y recomendaciones de como crear nuestro html, lo que nos servirá para poder obtener un mejor posicionamiento ante google. Con respecto a servidor utilizado fue Netlify, lo que me permitió crear el Deploy desde Github.
